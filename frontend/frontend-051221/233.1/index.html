Question 2

a. In your own words what will this point to and why. b. How can you fix this code.

const myObj = { name: "Timmy", greet: () => { console.log(`Hello ${this.name}`); }, }; myObj.greet();

when calling with an arrow function, 'this' will point to the global scope. We can change it to function declaration and it will fix things up.

const myObj = { name: "Timmy", greet() { console.log(`Hello ${this.name}`); } }; myObj.greet();
