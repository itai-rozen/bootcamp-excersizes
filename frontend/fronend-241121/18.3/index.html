If all elements share the same tagname, I can hold all of them together with 'document.querySelectorAll', loop over them with a forEach method and create variables dynamically.

Here's what i did:

document.querySelectorAll('div').forEach((div, i) => { window['div' + (i + 1)] = div })

now each div is in a different variable 'div1','div2' etc. i can also inject content & style in them the same way:

document.querySelectorAll('div').forEach((div, i) => {
window['div' + (i + 1)] = div
window['div' + (i+1)].textContent = `Yo! im div number ${i+1}`
window['div' + (i+1)].style.border = "1px solid brown"
})
